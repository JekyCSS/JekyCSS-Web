JekyCSS Documentation

Learn how to use JekyCSS, a lightweight and easy-to-use CSS framework to quickly design responsive websites.

Getting Started

To use JekyCSS in your project, include the following link in your HTML:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/JekyCSS/JekyCSS@0.1/jekycss.min.css">

Simply add the above line inside your <head> tag in your HTML file to start using JekyCSS.

Components

Buttons

JekyCSS provides reusable button components. To create a button, you can use the .btn class, and you can style it with the following modifier classes:

  • .bg-primary for a primary button with the default blue background.
  • .bg-secondary for a secondary button with a green background.

Here is an example:


<a href="#" class="btn bg-primary m-2">Primary Button</a>
<a href="#" class="btn bg-secondary m-2">Secondary Button</a>

Cards

JekyCSS provides a .card class to create card components. You can add padding and borders to them using the .p-2 class (padding of 16px) and .rounded for rounded corners.

Here is an example:


<div class="card p-2">
  <h4>Card Title</h4>
  <p>This is an example of a simple card component.</p>
</div>

Grid System

JekyCSS uses a flexible grid system with .row and .col-* classes to create responsive layouts. The columns are based on a 12-column system, and you can use classes like .col-4 to specify how many columns a particular element should take up.

Here is an example:


<div class="row">
  <div class="col-4">
    <div class="card p-2">
      <h4>Column 1</h4>
      <p>Content for the first column.</p>
    </div>
  </div>
  <div class="col-4">
    <div class="card p-2">
      <h4>Column 2</h4>
      <p>Content for the second column.</p>
    </div>
  </div>
  <div class="col-4">
    <div class="card p-2">
      <h4>Column 3</h4>
      <p>Content for the third column.</p>
    </div>
  </div>
</div>

Utility Classes

JekyCSS provides several utility classes for common styles:

  • .m-*: Margin classes to control the margin around elements (e.g., .m-1, .m-2, etc.).
  • .p-*: Padding classes to control the padding inside elements (e.g., .p-1, .p-2, etc.).
  • .text-center: Centers text horizontally inside an element.
  • .bg-primary: Applies the primary background color.

Here is an example:


<a href="#" class="btn bg-primary m-2">Example Button</a>
